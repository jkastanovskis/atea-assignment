#### Running the app

1. Run `docker-compose up --build -d` to build the container and run it in background;
2. The app should be available at http://localhost:5000 . The billing API endpoint is "/api/v1/Order/Process" . Project includes Swagger, which can be accessed via http://localhost:5000/swagger/index.html .
